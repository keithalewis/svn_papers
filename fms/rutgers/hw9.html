The Plackett family of copulas are given by \[ C_\theta(u,v) = \frac{1 + (\theta - 1)(u + v) - \sqrt{[1 + (\theta - 1)(u + v)]^2 - 4uv\theta(\theta - 1)}} {2(\theta - 1)} \] for \(\theta\ge0\) and \(C_1(u,v) = uv\). Note that \(C_0(u, v) = \max\{u + v - 1, 0\}\) and \(C_\infty(u, v) = \min\{u, v\}\) give the bookends of the Frech\'et-Hoeffding inequality.

Given two random variables \(X\) and \(Y\) the odds ratio is \(\theta(x,y) = P(X>x,Y>y)P(X\le x,Y\le y)/P(X>x,Y\le y)P(X\le x,Y>y)\). Note that if \(X\) and \(Y\) are independent, then \(\theta = 1\), if \(Y = X\) then \(\theta = \infty\), and if \(Y = -X\) then \(\theta = 0\) for all \(x\), \(y\).
