Your job is to provide a convincing argument that what you claim is correct.

The Heath-Jarrow-Morton model assumes that for each maturity, \(u\), the instantaneous forward rate satisfies the SDE \(df(t, u, \omega) = \alpha(t, u, \omega)\,dt + \sigma(t, u, \omega)\,dB_t\) for \(t\le u\), where \((B_t)\) is standard Brownian motion and \(\alpha\) and \(\sigma\) satisfy appropriate conditions. The price of a zero coupon bond at time \(t\) maturing at time \(u\) is \(D(t, u) = \exp(-\int_t^u f(t, s)\,ds\)). Recall the stochastic discount is \(D_t = \exp(-\int_0^t r_s\,ds)\) where \(r_t = f(t,t)\).

Exercise 1. Prove \(dD_t = -r_t D_t\,dt\).

Note \(d(e^{-X}) = e^{-X}\,dX + \frac{1}{2} e^{-X} (dX)^2\). Here \(X = \int_0^t r_s\,ds\) so \(dX = r_t\,dt\) and \((dX)^2 = 0\). We have \(dD_t = D_t(-r_t\,dt)\).

Exercise 2. Prove \(d(\int_t^u f(t,s)\,ds) = -f(t,t)\,dt + \int_t^u df(t,s)\, ds\).

We have \(\int_{t+dt}^u f(t + dt,s)\,ds - \int_t^u f(t,s)\,ds = \int_{t+dt}^u f(t,s) + df(t,s)\,ds - \int_t^u f(t,s)\,ds = \int_{t+dt}^t f(t,s)\,ds + \int_{t + dt}^u df(t,s)\,ds = - f(t,t)\,dt + \int_t^u df(t,s)\,ds\).

Exercise 3. Prove \(dD(t,u) = D(t,u)\{r_t\,dt - \int_t^u df(t,s)\,ds + \frac{1}{2}(\int_t^u \sigma(t,s)\,ds)^2\,dt\}\).

Here \(D(t,u) = e^{-X_t}\) with \(X_t = \int_t^u f(t, s)\,ds\) so \(dX = -f(t,t)\,dt + \int_t^u df(t,s)\,ds\) and \((dX)^2 = (\int_t^u \sigma(t,s)\,ds)^2\) since \(dB_u dB_v = 1(u = v)\,du\). We have \(dD(t,u) = D(t,u)\{f(t,t)\,dt - \int_t^u df(t,s)\, ds + \frac{1}{2}(\int_t^u \sigma(t,s)\,ds)^2\,dt\}\).

Exercise 4. Prove that \(t\mapsto D(t,u)D_t\) being a martingale implies \(\alpha(t,u) = \sigma(t,u)\int_t^u \sigma(t,s)\,ds\).

Note \(dD(t,u)D_t = D(t,u)\,dD_t + D_t\,dD(t,u) + dD(t,u)\,dD_t\) and that the last term is zero because the quadratic variation of \(D_t\) is zero. Hence \(dD(t,u)D_t = D(t,u)D_t(-r\,dt + f(t,t)\,dt - \int_t^u \alpha(t,s)\,dt + \sigma(t,s)\,dB_t\, ds + \frac{1}{2}(\int_t^u \sigma(t,s)\,ds)^2\,dt)\). An Ito process is a martingale if and only if the drift term vanishes so \(0 = -\int_t^u \alpha(t,s) + \frac{1}{2}(\int_t^u \sigma(t,s)\,ds)^2\). Differentiating with respect to \(u\) gives \(\alpha(t,u) = (\int_t^u \sigma(t,s)\,ds)\sigma(t,u)\).
