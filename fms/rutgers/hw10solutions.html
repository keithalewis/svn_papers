Exercise 1. For a fixed rate mortgate with rate \(r\) and periodic payment \(m\), the principal at time \(n\) satisfies \(P_{n+1} = (1 + r)P_n - m\). Find \(P_*\) such that \(P_* = (1 + r)P_* - m\). Define \(Q_n = P_n - P_*\) and show \(Q_{n+1} = (1 + r)Q_n\). If \(P_N = 0\), find the periodic payment \(m\) in terms of \(P_0\), \(r\), and \(N\).

We have \(P_* = m/r\) and \(Q_n = (1 + r)^nQ_0\) so \(0 = P_N = m/r + (1 + r)^N(P_0 - m/r)\) and \(m = P_0r/(1 - (1 + r)^{-N})\).

Exercise 2. Define \(r_q(t)\) by \(1/(1 + r_q(t)/q) = D(t, t + 1/q)\). Show \(\lim_{q\to\infty} q \log (1 + r_q(t)/q) = r_t\), the continuously compounded short rate process.

Note \(q\log(1 + r_q/q) = q\int_t^{t + 1/q} f(t,s)\,ds\) so \(\lim_{q\to\infty} q \log (1 + r_q(t)/q) = \lim_{h\to 0} (1/h) \int_t^{t + h} f(t,s)\,ds = f(t,t)\).

Exercise 3. Define \(\bar{r}_h(t) = r_{1/h}(t)\). Show \(\lim_{h\to 0} d\bar{r}_h(t)/dh = (f(t,t)^2 + f_2(t,t))/2\), where \(f_2(t, s)\) is the partial derivative of the instantaneous forward with respect to the second variable.

Since \(\bar{r}_h = (e^{\int_t^{t+h} f(t,s)\,ds} - 1)/h\) we have \(d\bar{r}_h/dh = [(e^{\int_t^{t+h} f(t,s)\,ds} - 1) - h(e^{\int_t^{t+h} f(t,s)\,ds} f(t, t + h))]/h^2\). Apply L'Hopital's rule to get \begin{aligned} \lim_{h\to 0} \frac{d\bar{r}_h}{dh} &= \lim_{h\to 0} \frac{e^\int f(t, t + h)^2 h + e^\int f_2(t,t+h) h + e^\int f(t,t+h) - e^\int f(t, t + h)}{2h}\\ &= (f(t,t)^2 + f_2(t,t))/2 \end{aligned}

Exercise 4. In section 5.4 of the Kalotay, Yang, and Fabozzi paper, assume \(p_0(l) = \lambda e^{-\lambda l}\), \(l\ge0\). Given \(f_0\) and \(f\), find \(l_0\) and \(p(l)\). Why is this probably not a good choice for the laggard distribution?

Solving \((f_0 - f)/f_0 = \int_0^{l_0} p_0(l)\,dl = 1 - e^{-\lambda l_0}\) for \(l_0\) gives \(l_0 = (1/\lambda)\log f_0/f\). The new laggard distribution is the same as the old one. This is not good because it does not model burnout.
