Measurable Functions

A function\(X\colon\Omega\to\R\) is \(\cal{A}\) measurable if \(X^{-1}((-\infty, x])\in\cal{A}\) for all \(x\in\R\).

Exercise. Show the smallest algebra of \(\R\) containing the sets \((-\infty, x]\), \(x\in\R\) consists of sets that are finite disjoint unions of intervals of the form \((a, b]\), where \(a,b\in\R\) or \(a = -\infty\).

Exercise. Show that if the function \(X\) takes on only a finite number of values \(\{x_i\}\) then then \(X^{-1}(\{x_i\}) = X^{-1}((x_i - \epsilon, x_i])\) for some \(\epsilon\gt 0\).

Exercise. Show \(X\) is \(\cal{A}\) measurable if and only if \(X\) is constant on the atoms of \(\cal{A}\).

Another way of saying this is that \(X\) depends only on the partial information provided by \(\cal{A}\). If you only know the atom containing \(\omega\), then you know the value of \(X(\omega)\).
