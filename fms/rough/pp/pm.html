Probability Measures

We only consider finitely additive probability measures to avoid the technical issues invovled with countably additive measures.

A probability measure is a function \(P\colon\mathcal{A}\to[0,1]\) such that \(P(\Omega) = 1\) and \(P(A\cup B) = P(A) + P(B)\) if \(A\) and \(B\) are disjoint (\(A\cap B = \emptyset\)).

Exercise. Show \(P(\Omega\setminus A) = 1 - P(A)\).

Exercise. Show \(P(A\cup B) = P(A) + P(B) - P(A\cap B)\).

Exercise. Show \(P(\cup_{i=1}^n A_i) = \sum_i P(A_i) - \sum_{i\lt j} P(A_i\cap A_j) + \sum_{i\lt j\lt k} P(A_i\cap A_j\cap A_k) - \cdots \pm P(\cap_i A_i)\).

Exercise. Show that if \(g(A) = \sum_{B\subseteq A} f(B)\) then \(f(B) = \sum_{A\subseteq B} (-1)^{|B| - |A|} g(A)\) where \(|A|\) is the cardinality of the set \(A\).

Remarks

What is a probability? Is it frequency of occurence? Is it a degree of belief? Can it be represented by a number between 0 and 1?
