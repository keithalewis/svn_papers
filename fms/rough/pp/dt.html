Decision Theory

How to make an optimal decision based on information.

  1. Enumerate the sample space \(\{\omega_j\}\).
  2. Assign prior probabilities \(p(\omega_j|x)\) which maximize the entropy given prior information \(x\).
  3. Incorporate new evidence \(E\) using Bayes' theorem to get posterior probabilities \(p(\omega_j|x,E)\).
  4. Enumerate possible decisions \(\{D_i\}\).
  5. Specify a loss function \(L(D_i,\omega_j)\).
  6. Make decision \(D_i\) that minimizes the expected loss \begin{align*} <L>_i = \sum_j L(D_i,\omega_j)p(\omega_j|x,E) \end{align*}
