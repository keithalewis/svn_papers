Conditional Expectation

In elementary probability theory the conditional expectation of event \(A\) given event \(B\) is defined as \(P(A|B) = P(A\cap B)/P(B)\). This localizes the sample space to the set \(B\) and satisfies \(P(B|B) = 1\).

Given a random variable \(X\) and an algebra \({\cal A}\), the conditional expectation of \(X\) given \({\cal A}\), \(E[X|{\cal A}]\), is an \({\cal A}\) measurable random variable that satisfies \[ \int_A E[X|{\cal A}]\,dP = \int_A X\,dP, A\in{\cal A}. \] The conditional expectation is the Radon-Nikodym derivative of the measure \(Q\) defined by \(Q(A) = \int_A X\,dP\) with respect to \(P\).

Exercise. Show the conditional expectation of a random variable with respect to the trivial algebra is the standard expectation of the random variable.

Exercise. (Tower property.) If \({\cal A}\subseteq{\cal B}\) then \(E[X|{\cal A}] = E[E[X|{\cal B}]|{\cal A}]\).

Exercise. If \(\{A_i\}\) is a partition show \[ E[X|{\cal A}](\omega) = \frac{1}{P(A_\omega)}\int_{A_\omega} X\,dP, \] where \(A_\omega\) is the atom containg \(\omega\) and \({\cal A}\) is the algebra generated by the partition.
