Probability Preliminaries

A rough guide to what you should know about probability.

Sample Space: the set of all possible outcomes.

Event: a subset of the sample space.

Algebra: the mathematical way to represent partial information about outcomes using events.

Probability Measure: a function that assigns probabilities to events.

Probabilty Space: a triple \(\langle P,\Omega,\cal{A}\rangle\).

Measurable Function: keeps you honest about partial information.

Random Variable: a placeholder for a number with additional information about the probability of the values it can take.

Cumulative Distribution Function : a function determining everything there is to know about a random variable.

Conditional Expectation: reflects our best estimate of a random variable given partial information.

Maximum Entropy Principle: Generalization of Laplace's principle of indifference.

Decision Theory: How to make an optimal decision.

Joint Distribution: is a function determining everything there is to know about two or more random variables.

Independence: means multiply.

Stochastic Process: is a collection of random variables usually indexed by time.

Lévy Process: is a stochastic process characterized by its increments being independent and stationary.

Brownian Motion: is a Lévy process with normally distributed increments.

Poisson Process: the simplest Lévy process.
